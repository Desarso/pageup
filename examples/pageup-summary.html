Rules of the road

Small surface. Clear promises.

Pageup is intentionally narrow: it publishes HTML reliably and leaves the creative range wide open.

Private publishing
Only explicitly authorized Ed25519 keys can upload. Installing the CLI alone grants no access.
Public viewing
Anyone with the generated link can view a page. There is no public index, but links are not access-controlled.
One project, one lasting ID
Every new upload gets a UUIDv7. Its creator or an admin can publish later revisions—or switch between one page and a mini-site—under that same ID.
File, folder, or raw HTML
Point the CLI at an HTML file, a directory of up to 100 HTML pages, or pipe generated markup to standard input. Each upload may contain up to 5 MiB of HTML.
Unlimited projects
There is no configured page-count limit. The practical boundary is the persistent storage available on the server.
